# Whisper Training Skeleton

This folder contains guarded training scaffolding for a future phase. Phase 9 does not train or fine-tune Whisper automatically.

Use this only to preview the planned configuration:

```powershell
python training/train_whisper_skeleton.py --config configs/whisper_finetune_config.yaml
```

Do not commit checkpoints, model weights, optimizer states, or generated trainer logs.
