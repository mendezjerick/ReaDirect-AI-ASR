# CMUdict

Place CMU Pronouncing Dictionary files here:

- `cmudict.dict`
- `cmudict.phones`
- `cmudict.symbols`

The Phase 2 phoneme loader parses these files locally and uses them to enrich ReaDirect expected answers with simple ARPABET metadata. Keep license/readme information with the dataset when available.

CMUdict contains pronunciation dictionary data, not learner audio.
