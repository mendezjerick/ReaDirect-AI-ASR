# Enriched Content Bank

This directory is for generated ReaDirect content-bank enrichment outputs.

Tracked files:

- This README.
- `.gitkeep` placeholders.

Ignored by default:

- Enriched CSVs.
- Generated JSON files.
- Generated reports.
- Import-ready ZIP files.

Do not overwrite original content-bank files. Generated outputs should be reviewed before any import back into the main Laravel ReaDirect repository.
