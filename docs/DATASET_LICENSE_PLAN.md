# Dataset License Plan

## Active Commercial-Safe Direction

- ReaDirect curated CSV content bank.
- CMUdict phoneme dictionary.
- Speechocean762 pronunciation-scoring dataset.

These are the only active datasets in the current AI/ASR pipeline.

## Removed From Active Plan

- L2-ARCTIC: research-only optional, non-commercial only. Its CC BY-NC 4.0 terms make it unsuitable for deployable model training, production inference, or commercial/government-client evaluation unless a separate compatible license is obtained.
- PF-STAR: future optional only because it requires access/request.

## Rule

Always check dataset licenses before using data to train deployable models, run production inference, or produce client-facing evaluation results.
